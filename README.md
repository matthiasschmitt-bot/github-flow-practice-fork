# Übungsprojekt: GitHub Flow (Fork)

Dieses Repository ist ein **Fork**, der genutzt wird, um den GitHub Flow zu üben – also das Arbeiten mit Branches, Commits, Pull Requests und Merges.

## Inhalte
- Testdateien aus dem Original-Repository
- eigene Änderungen und Experimente
- Arbeiten mit Forks und Pull Requests
- Versionskontrolle und Branch-Strategien

## Ziel
Verstehen, wie Forks funktionieren und wie man Änderungen strukturiert zurück in ein Original-Repository einbringen kann.
